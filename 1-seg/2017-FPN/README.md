## 1. 简介

### 1.1 简介

**特征图金字塔网络FPN**（**Feature Pyramid Networks**）是2017年提出的一种网络，



FPN要解决的是物体检测中的多尺度问题，通过简单的网络连接改变，在基本不增加原有模型计算量的情况下，大幅度提升了小物体检测的性能。
